## 发布到 GitHub

### 步骤 1：创建 GitHub 仓库

1. 访问 https://github.com/new
2. 仓库名称：`ppt-image-replacer`
3. 描述：`A visual tool for batch replacing images in PowerPoint files`
4. 选择 Public
5. **不要**勾选 "Add a README file"、"Add .gitignore" 或 "Choose a license"
6. 点击 "Create repository"

### 步骤 2：推送代码

在项目目录中运行以下命令：

```bash
# 配置 Git 用户信息
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

# 添加远程仓库
git remote add origin https://github.com/lesliehanx/ppt-image-replacer.git

# 重命名主分支
git branch -M main

# 推送代码
git push -u origin main
```

### 步骤 3：创建 Release

1. 访问 https://github.com/lesliehanx/ppt-image-replacer/releases
2. 点击 "Create a new release"
3. 标签版本：`v1.0.0`
4. 发布标题：`PPT Image Replacer v1.0.0`
5. 描述：
   ```
   ## 功能特点
   - 可视化界面，按页分组显示图片
   - 支持单张替换、多选替换和批量替换
   - 支持扫描组合形状中的图片
   - 保持图片位置和大小不变
   ```
6. 点击 "Publish release"

## 发布到 PyPI

### 步骤 1：安装打包工具

```bash
pip install build twine
```

### 步骤 2：构建包

```bash
python -m build
```

### 步骤 3：上传到 PyPI

```bash
twine upload dist/*
```

或者使用 GitHub Actions 自动发布：

1. 在 GitHub 仓库设置中添加 Secret：
   - 名称：`PYPI_API_TOKEN`
   - 值：你的 PyPI API Token

2. 创建 Release 时会自动触发发布工作流

## 安装使用

### 从 PyPI 安装

```bash
pip install ppt-image-replacer
```

### 从 GitHub 安装

```bash
pip install git+https://github.com/lesliehanx/ppt-image-replacer.git
```

### 直接运行

```bash
git clone https://github.com/lesliehanx/ppt-image-replacer.git
cd ppt-image-replacer
pip install -r requirements.txt
python ppt_image_replacer.py
```

## 版本更新

1. 修改 `ppt_image_replacer.py` 中的 `__version__`
2. 修改 `pyproject.toml` 中的 `version`
3. 修改 `setup.py` 中的 `version`
4. 提交更改：
   ```bash
   git add .
   git commit -m "Bump version to 1.1.0"
   git tag v1.1.0
   git push origin main --tags
   ```
5. 在 GitHub 创建新的 Release
