## 核心文件说明

### 主程序
- `ppt_image_replacer.py` - 可视化GUI界面程序

  - `SlideCard` 类：幻灯片页面卡片组件（含全选功能）
  - `PPTImageReplacer` 类：主程序类

### 配置文件
- `requirements.txt` - Python依赖包列表
- `run_ppt_tool.bat` - Windows启动脚本

### 文档

- `使用说明.md` - 详细使用说明
- `项目结构.md` - 项目文件结构说明
- `项目总结.md` - 项目完成情况总结

### 测试文件
- `test_ppt.py` - 创建包含图片的测试PPT

## 代码架构

### ImageCard 类
单张图片的卡片组件，包含：
- 勾选框（Checkbutton）
- 缩略图预览（180x180像素）
- 图片名称
- 状态标签（原图/已替换）
- 选择新图片按钮

### SlideCard 类
幻灯片页面的卡片组件，包含：
- 页面标题
- 图片数量
- 全选本页按钮
- 多个 ImageCard 组件
- 横向滚动支持

### PPTImageReplacer 类
主程序类，包含：
- PPT文件选择和扫描
- 图片文件夹选择
- 图片勾选管理
- 多选替换功能
- 批量替换功能
- PPT保存功能

## 主要功能

### 1. 单张替换
- 点击任意图片的"选择新图片"按钮
- 选择新图片后立即更新缩略图

### 2. 多选替换
- 勾选需要替换的图片（可跨页多选）
- 点击"替换勾选的图片"按钮
- 选择一张新图片
- 所有勾选的图片都会被替换为这张图片

### 3. 批量替换
- 选择图片文件夹
- 图片按顺序自动匹配
- 点击"一键替换全部"按钮

## 快速开始

1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. 运行程序：
   ```bash
   python ppt_image_replacer.py
   # 或双击 run_ppt_tool.bat
   ```

3. 按照界面提示操作
