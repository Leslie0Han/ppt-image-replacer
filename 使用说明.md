# PPT 图片批量替换工具 - 使用说明

## 解决的痛点

做PPT时经常需要插入大量图片（户型图、总图等），但每轮方案修改后都要重新手动导入图片。这个工具可以一键替换PPT中的所有图片，保持位置和大小不变。

## 安装

```bash
pip install -r requirements.txt
```

## 使用方法

### 1. 启动工具

双击 `run.bat` 或运行：
```bash
python ppt_image_replacer.py
```

### 2. 操作步骤

1. **选择PPT文件** - 点击"选择PPT"按钮，选择要更新的 `.pptx` 文件
2. **选择图片文件夹** - 点击"选择文件夹"按钮，选择包含新图片的文件夹
3. **扫描PPT图片** - 点击"扫描PPT图片"，工具会列出PPT中的所有图片
4. **匹配图片** - 点击"匹配图片"，工具会自动匹配图片
5. **一键替换** - 点击"一键替换"，生成新的PPT文件

### 3. 图片匹配规则

工具支持多种匹配方式（按优先级）：

1. **精确匹配** - 新图片文件名与PPT中的图片名称完全相同
2. **包含匹配** - 文件名相互包含
3. **顺序匹配** - 如果图片数量相同，按顺序匹配
4. **索引匹配** - 按图片在PPT中的顺序匹配（兜底方案）

### 4. 输出结果

- 替换后的PPT文件会保存在原文件同目录，文件名添加 `_updated` 后缀
- 原文件不会被修改

## 示例

假设你的PPT中有以下图片：
- 第1页：户型图
- 第2页：总图
- 第3页：效果图

将新图片放在一个文件夹中，按顺序命名：
```
新图片/
├── 01_户型图.png    (更新后的户型图)
├── 02_总图.jpg      (更新后的总图)
└── 03_效果图.png    (更新后的效果图)
```

选择这个文件夹，工具会按顺序匹配并替换。

## 支持的图片格式

- PNG (.png)
- JPEG (.jpg, .jpeg)
- GIF (.gif)
- BMP (.bmp)
- TIFF (.tiff)

## 注意事项

- 只支持 `.pptx` 格式（不支持旧版 `.ppt`）
- 替换后会生成新文件，原文件保持不变
- 建议替换前备份原文件
- 图片会保持在PPT中的原始位置和大小
