# PPT Image Replacer

一个可视化的PPT图片批量替换工具，让你可以直观地看到每页的图片，并支持单张替换、多选替换和批量替换。

[![Python](https://img.shields.io/badge/Python-3.7+-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

## 解决的痛点

做PPT时经常需要插入大量图片，但每轮方案修改后都要重新手动导入图片。这个工具提供可视化界面，让你可以：

- 直观地看到每页有哪些图片
- 通过缩略图快速识别图片内容
- 灵活选择单张替换、多选替换或批量替换
- 支持扫描组合形状中的图片（打组的图片）

## 功能特点

- **可视化界面**：按页分组显示图片，每页一个卡片
- **缩略图预览**：直接看到PPT中的图片内容（180x180像素）
- **单张替换**：点击任意图片单独选择新图片替换
- **多选替换**：勾选多张图片，一次性替换为同一张新图片
- **批量替换**：选择文件夹按顺序替换所有图片
- **支持组合图片**：可以扫描PPT中打组的图片
- **保持位置**：替换后图片位置和大小不变
- **安全保存**：生成新文件，不修改原文件

## 安装

### 方法一：使用 pip 安装（推荐）

```bash
pip install ppt-image-replacer
```

### 方法二：从源代码安装

```bash
git clone https://github.com/lesliehanx/ppt-image-replacer.git
cd ppt-image-replacer
pip install -e .
```

### 方法三：直接运行

```bash
git clone https://github.com/lesliehanx/ppt-image-replacer.git
cd ppt-image-replacer
pip install -r requirements.txt
python ppt_image_replacer.py
```

## 使用方法

### 启动程序

如果使用 pip 安装：
```bash

```

如果直接运行：

或双击 `run_ppt_tool.bat`（Windows）

### 操作步骤

#### 方法一：单张替换

1. 点击"选择PPT"按钮选择PPT文件
2. 点击"扫描"按钮查看每页的图片
3. 点击任意图片下方的"选择新图片"按钮
4. 选择新图片
5. 点击"保存PPT"生成新文件

#### 方法二：多选替换（适合不同页面的相同图片）

1. 点击"选择PPT"按钮选择PPT文件
2. 点击"扫描"按钮
3. 勾选需要替换的图片（可跨页多选）
4. 点击"替换勾选的图片"按钮
5. 选择一张新图片，所有勾选的图片都会被替换
6. 点击"保存PPT"生成新文件

#### 方法三：批量替换

1. 点击"选择PPT"按钮选择PPT文件
2. 点击"扫描"按钮
3. 点击"选择文件夹"按钮选择图片文件夹
4. 图片会按顺序自动匹配
5. 点击"一键替换全部"按钮
6. 点击"保存PPT"生成新文件

## 支持的格式

### 图片格式
- PNG (.png)
- JPEG (.jpg, .jpeg)
- GIF (.gif)
- BMP (.bmp)
- TIFF (.tiff)

### PPT格式
- PowerPoint (.pptx)

## 故障排除

### 问题1：提示"Python not found"

**解决方案：**
1. 安装 Python 3.7 或更高版本：https://www.python.org/downloads/
2. 安装时勾选 "Add Python to PATH"
3. 安装完成后重启命令提示符

### 问题2：提示缺少模块

**解决方案：**
```bash
pip install -r requirements.txt
```

### 问题3：组合图片扫描不到

**说明：** 工具已支持递归扫描组合形状中的图片，如果仍然扫描不到，请检查：
1. 图片是否被锁定或隐藏
2. 图片是否在母版中
3. 图片格式是否受支持
