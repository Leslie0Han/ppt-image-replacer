# PPT 图片批量替换工具 - 项目总结

## 项目完成情况

已完成一个完整的PPT图片批量替换工具，解决了以下痛点：

**核心问题**：做PPT时需要插入大量图片（户型图、总图等），每轮方案修改后都要重新手动导入图片。

**解决方案**：开发一个可视化工具，可以直观地看到每页的图片，并支持单张替换、多选替换和批量替换。

## 功能实现

### 已实现功能

1. **可视化界面** - 卡片式布局，每页显示一栏
2. **缩略图预览** - 直接看到PPT中的图片内容（180x180像素）
3. **单张替换** - 点击任意图片单独选择新图片替换
4. **多选替换** - 勾选多张图片，一次性替换为同一张新图片
5. **批量替换** - 选择文件夹按顺序替换所有图片
6. **一键替换** - 批量替换所有已选择的图片
7. **居中布局** - 图片卡片居中显示，充分利用空间
8. **支持组合图片** - 可以扫描PPT中打组的图片（递归遍历）
9. **结果保存** - 生成新文件，不修改原文件

### 技术实现

- **Python + tkinter** - GUI 界面
- **python-pptx** - PPT文件处理
- **Pillow** - 图片处理和缩略图生成
- **Grid布局** - 图片卡片居中显示
- **Checkbutton** - 多选功能
- **递归算法** - 遍历组合形状中的图片

## 使用方法

### 方法一：单张替换

1. 选择PPT文件
2. 点击"扫描"按钮
3. 查看每页的图片缩略图
4. 点击任意图片的"选择新图片"按钮
5. 选择新图片
6. 点击"保存PPT"

### 方法二：多选替换（适合不同页面的相同图片）

1. 选择PPT文件
2. 点击"扫描"按钮
3. 勾选需要替换的图片（可跨页多选）
4. 点击"替换勾选的图片"按钮
5. 选择一张新图片
6. 所有勾选的图片都会被替换为这张图片
7. 点击"保存PPT"

### 方法三：批量替换

1. 选择PPT文件
2. 点击"扫描"按钮
3. 选择图片文件夹（图片按顺序匹配）
4. 点击"一键替换全部"
5. 保存PPT

## 故障排除

### 问题1：双击无反应或闪退

**解决方案：**
1. 右键点击 `run_ppt_tool.bat`，选择"以管理员身份运行"
2. 或者打开命令提示符，手动运行：
   ```bash
   cd "D:\leslie\60_claude项目库\PPT工具"
   python ppt_image_replacer.py
   ```

### 问题2：提示"Python not found"

**解决方案：**
1. 安装 Python 3.7 或更高版本：https://www.python.org/downloads/
2. 安装时勾选 "Add Python to PATH"
3. 安装完成后重启命令提示符

### 问题3：提示缺少模块

**解决方案：**
```bash
pip install -r requirements.txt
```

### 问题4：组合图片扫描不到

**说明：** 工具已支持递归扫描组合形状中的图片，如果仍然扫描不到，请检查：
1. 图片是否被锁定或隐藏
2. 图片是否在母版中
3. 图片格式是否受支持
